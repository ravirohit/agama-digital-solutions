# Agama Web Designer

A static website for Agama Web Designer, a company specializing in web design and standalone applications.

## Features

- Responsive design
- Clean, professional layout
- Services showcase
- Contact form
- Smooth scrolling navigation

## Technologies Used

- HTML5
- CSS3
- JavaScript

## How to View

Simply open `index.html` in your web browser.

## Structure

- `index.html` - Main HTML file
- `css/style.css` - Stylesheet
- `js/script.js` - JavaScript for interactivity
